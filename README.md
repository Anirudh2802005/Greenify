# 🌱 Greenify - Plastic Waste Management App

## 📱 Description
Greenify is a Flutter-based mobile application that helps users track plastic usage and promotes eco-friendly habits.

## 🚀 Features
- User Login & Signup (Firebase)
- Dashboard
- Plastic usage calculator
- Awareness content
- User profile

## 🛠️ Tech Stack
- Flutter
- Dart
- Firebase Authentication

## ▶️ How to Run
```bash
flutter pub get
flutter run
